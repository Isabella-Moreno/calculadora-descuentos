## Funcionalidades
- **Descuento 5%:** Aplica un descuento del 5% al precio ingresado
- **Descuento 10%:** Aplica un descuento del 10% al precio ingresado
- **Descuento 15%:** Aplica un descuento del 15% al precio ingresado
- **Comparar descuentos:** Compara el precio final con los tres descuentos

## Instrucciones de ejecución
1. Clonar el repositorio
2. Abrir en IntelliJ IDEA
3. Ejecutar `Main.java`
4. Ingresar el precio cuando lo solicite la consola

## Flujo colaborativo
- Cada funcionalidad fue desarrollada en una rama independiente
- Se crearon Pull Requests para cada funcionalidad
- Cada PR fue revisado y aprobado por al menos un compañero
- Se hizo merge hacia main después de la aprobación
